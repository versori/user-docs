## Embedded Connector

Setting a connector as embedded will enable a multi tenanted integration. A multi tenanted integration allows external
non Versori users to connect with their own credentials to a connector.
[See multi tenancy use case](../use-cases/multi-tenancy)

To enable an embedded connector, select the authentication method needed. That‘s it!
