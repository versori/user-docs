---
title: "users"
description: "Manage users in your organisation."
---

The `users` command (alias: `user`) manages users at the organisation level.

<Note>
To manage users linked to a specific project, use [`projects users`](/latest/cli/commands/projects#versori-projects-users) instead.
</Note>

## Subcommands

| Subcommand | Description |
|---|---|
| `list` | Lists users in the current organisation context |
| `create` | Create a new user in the current organisation |

---

### `versori users list`

Lists all users in the current organisation context.

```sh
versori users list
```

---

### `versori users create`

Create a new user in the current organisation.

```sh
versori users create --display-name <name> --external-id <id> [flags]
```

**Flags:**
* `-n`, `--display-name`: Display name of the user
* `-e`, `--external-id`: External ID of the user
