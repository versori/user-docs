---
title: "execution-pools"
description: "Manage execution pools for your organisation."
---

The `execution-pools` command (alias: `ep`) manages execution pools.

## Subcommands

| Subcommand | Description |
|---|---|
| `list` | Lists available execution pools for the current organisation |

---

### `versori execution-pools list`

Lists available execution pools for the current organisation.

```sh
versori execution-pools list
```
